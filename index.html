Loops in Python

In Python, **loops** allow you to repeatedly execute a block of code as long as a certain condition is true. There are two main types of loops in Python: **`for` loops** and **`while` loops**.

for` loop:

The `for` loop is used to iterate over a sequence (like a list, tuple, string, or range) and execute a block of code for each item in that sequence.

  • Syntax:

    • for item in sequence:
    # Code block to execute for each item in the sequence

    fruits = ["apple", "banana", "cherry"]
    for fruit in fruits:
    print(fruit)

    Output: apple
    banana
    cherry

    `while` loop:

    The `while` loop repeats as long as a given condition is true. It is generally used when you do not know the number of iterations in advance.

  • Syntax:

    • while condition:
    # Code block to execute as long as the condition is true

    x = 0
    while x < 5:
    print(x)
    x += 1 # Increment x by 1

    Output:
    0 1 2 3 4

    Controlling Loops with `break`, `continue`, and `else`:

  • `break`: Exits the loop entirely, even if the loop condition hasn't been met.
  • `continue`: Skips the current iteration and moves to the next iteration of the loop.
  • `else`: Can be used with both `for` and `while` loops. The code inside the `else` block will run when the loop is not terminated by a `break` statement (i.e., when the loop runs to completion).

    • for i in range(10):
    if i == 5:
    break # Exits the loop when i equals 5
    print(i)

    Output:
    0 1 2 3 4

    for i in range(10):
    if i == 5:
    continue # Skips the iteration when i equals 5
    print(i)

    Output:
    0 1 2 3 4 6 7 8 9

    for i in range(5):
    print(i)
    else:
    print("Loop finished successfully")

    Output:
    0 1 2 3 4
    Loop finished successfully

    x = 0
    while x < 5:
    print(x)
    x += 1
    else:
    print("While loop completed")

    Output:
    0 1 2 3 4
    While loop completed
